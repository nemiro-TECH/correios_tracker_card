# Correios Tracker Card

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)

Card Lovelace para o [Correios Tracker](https://github.com/nemiro-TECH/correios-tracker-ha).

## Instalação via HACS

1. HACS → Interface → ⋮ → Repositórios personalizados
2. URL: `https://github.com/nemiro-TECH/correios-tracker-card` | Tipo: Lovelace
3. Instalar → Limpar cache do navegador

## Uso

```yaml
type: custom:correios-tracker-card
```

## Funcionalidades

- Lista todos os pacotes rastreados
- Badge **Entregue** / **Em trânsito**
- Botão **+ Adicionar** pacote direto no dashboard
- Botão **✕** para remover pacote
